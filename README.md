# Установка Flutter SDK

Для начала работы нужно установить сам Flutter SDK с оффициального сайта: [ссылка](https://storage.googleapis.com/flutter_infra_release/releases/stable/windows/flutter_windows_3.38.2-stable.zip)

После скачивания архива, нужно распаковать его в какую-нибудь папку. ОБЯЗАТЕЛЬНО: путь к папке не должен содержать кириллицу или спец. символов. 

После нужно добавить Flutter в переменную PATH, чтобы зарабатали комманды для компиляции кода. Для этого:


1. Кликаем ПКМ по "этот компьютер"
2. Кликаем свойства
3. Выбираем "дополнительные параметры системы"
4. Кликаем "Переменные среды"
5. Далее, создаем в графе "переменные среды пользователя ... " переменную. Называем ее как угодно, но для примера назовем flutterSDKBin, и в значении выбираем путь до папки bin нашего flutter. 
6. Нажимаем ОК
7. Ищем в графе "системные переменные" переменную Path, кликаем по ней лкм два раза и в самом низу вписываем еще раз путь до папки bin нашего flutter. 
8. Нажимаем ОК
9. Нажимаем ОК в окне "переменные среды"
10. Последний раз нажимаем ОК в окне "свойства системы"

После этого закрываем все открытые терминалы или cmd. И открываем заново. Для проверки, что все установилось корректно вводим комманду: 

```1

flutter --version
dart --version

```
Если никаких ошибок не появилось то поздравляю, FlutterSDK успешно был установлен на вашу систему.

# Установка Flutter в VSC

Для установки Flutter в VSC для начала открываем VSC и в левой панели выбираем **Extensions** и в меню поиска пишем **Flutter**. Нас интересует самый первый, от Dart Code. Устанавливаем его и ждем пока VSC сообщит об полной успешной установки расширения

Далее, Нажимаем сочетание клавиш: 
```
CTRL + SHIFT + P 
```
И в открывшейся поисковой строке пишем: Flutter и выбираем: **Flutter: New  Project**. После чего, VSC напишет что не установлен Dart и сам предложит его установить (будет синяя кнопка Install SDK) нажимаем на нее и выбираем куда установиться наш SDK. ВАЖНО!!!! Не выбираем ту же папку с flutter куда мы устанавливали ранее, нужно создать именно отдельную папку куда и установиться все необходимое. 

После выбора нажимаем ОК и ждем установки. По ее окончании будет еще одна синяя кнопка: **Add to the Path** нажимаем и после этого закрывем вновь все терминалы и консоли, а также сам VSC. 

Теперь можно пользоваться Flutter в полной мере!
